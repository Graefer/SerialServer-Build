Der Serial WebSocket Server läuft jetzt im Hintergrund.

Schnellstart

Server-Status prüfen im Terminal:

curl http://localhost:8765/health

Logs anzeigen:

tail -f /usr/local/serialserver/logs/server.log

Wichtige Dateien

  • Konfiguration: /usr/local/serialserver/config.json
  • Logs: /usr/local/serialserver/logs/
  • Server-Script: /usr/local/serialserver/server.py

Service-Verwaltung

Status prüfen:

sudo launchctl list | grep serialserver

Neu starten:

sudo launchctl kickstart -k system/com.serialserver

Stoppen:

sudo launchctl unload /Library/LaunchDaemons/com.serialserver.plist
