Installation

Der Installer führt folgende Schritte durch:

  1. Kopiert Programmdateien nach /usr/local/serialserver
  2. Erstellt Python Virtual Environment
  3. Installiert benötigte Python-Pakete (Flask, PySerial)
  4. Richtet LaunchDaemon für automatischen Start ein
  5. Startet den Service

Nach der Installation

Der Server läuft im Hintergrund. Serial Ports werden erst geöffnet wenn die PVS-App eine WebSocket-Verbindung herstellt.

Endpunkte:

  • HTTP-API: http://localhost:8765
  • WebSocket: ws://localhost:8765/ws
  • Health-Check: http://localhost:8765/health

Konfiguration

Passen Sie die Geräte-Konfiguration an in:

/usr/local/serialserver/config.json

Nach Änderungen Service neu starten:

sudo launchctl kickstart -k system/com.serialserver
