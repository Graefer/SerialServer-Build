Willkommen zum Serial WebSocket Server

Dieser Assistent installiert den Serial WebSocket Server auf Ihrem Mac.

Features

Automatische Erkennung von USB-Serial und Bluetooth-Geräten
WebSocket Push-Benachrichtigungen in Echtzeit
REST-API für einfache Integration
Automatischer Start bei Systemstart (On-Demand Serial Port)
Automatischer Reconnect bei Verbindungsabbruch

Systemanforderungen

  • macOS 10.15 (Catalina) oder neuer
  • Python 3.7 oder neuer (vorinstalliert auf modernen Macs)
  • Ca. 50 MB freier Speicherplatz

Hinweis: Die Installation benötigt Administrator-Rechte.
